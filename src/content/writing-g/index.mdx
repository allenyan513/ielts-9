# 雅思写作G类总结

## Writing Task 1

在雅思G类（General Training）考试中，**Writing Task 1** 通常要求你写一封**信件**，而这些信件的主题覆盖日常生活中会遇到的各种情境。以下是常见的**写作题目主题分类和具体示例**，方便你备考时有针对性地练习：

1. 投诉类
2. 请求帮助类
3. 提供建议类
4. 感谢类
5. 道歉类
6. 邀请类
7. 求职与工作类
8. 信息通知类
9. 推荐与介绍类

## Writing Task 2
在 **雅思G类写作（Writing Task 2）** 中，任务是写一篇**议论文（essay）**，和A类考试相同，考查的是你的**观点表达、论证能力、语言准确性和逻辑性**。但G类话题通常更加**贴近生活、实用性强、社会性强**，较少涉及学术性深奥话题。

1. 教育类
2. 工作与职业类
3. 科技与社会
4. 政府与社会责任
5. 家庭与人际关系
6. 环境类
7. 消费与广告类
8. 健康与生活方式
9. 老龄化与人口问题
10. 文化与旅游
